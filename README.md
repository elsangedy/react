# Reac

## Installation Guide

Enter the following commands in the terminal

```bash
$ git clone https://github.com/munir7/react.git
$ cd react
$ npm install
$ webpack -w
```


### Prerequisites

* Must have [Git](http://git-scm.com/) installed

* Must have [node.js (at least v0.10.0)](http://nodejs.org/) installed with npm (Node Package Manager)

* Must have [Webpack](http://webpack.github.io/) node package installed globally
